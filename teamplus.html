Teamplus資料同步說明     校務系統通知說明

資料庫IP: 203.64.37.44 資料庫名稱 tp_sync


1.團隊(部門/科系/班級)同步：

資料表名稱 group_sync
id 主鍵,數字,免填
parent 字串,父部門代碼.沒有父部門則空白
code 字串,部門代碼.例如106000例如ND31T
name 字串,部門名稱.例如護理系二一甲
admin 字串,團隊長的帳號.例如:Q221300900
members 字串,團隊成員的帳號,以逗號分隔. 例如:Q222333900,A121311900
is_delete boolean, 如果要刪除這個團隊才填true , 正常填false

2.學生同步：

資料表名稱 student_update_records
id 主鍵,數字,免填
name 字串,學生姓名
number 字串,學號. 例如 s10643037 (這個欄位資料將作為Teamplus App的帳號)
department 字串,班級代碼.例如ID41A
email 字串,例如 s10643037@ems.tcust.edu.tw
password 字串,密碼.例如:870512
status 數字.此帳號的狀態. 1=> 正常使用 0=>停用此帳號

3.職員/教師同步：

資料表名稱 staff_update_records
id 主鍵,數字,免填
name 字串,職員/教師姓名
number 字串,身分證號. 例如 A101447820 (這個欄位資料將作為Teamplus App的帳號)
department 字串,單位代碼.例如 110000
email 字串,例如 ma00@tcust.edu.tw
password 字串,密碼.例如:870512
status 數字.此帳號的狀態. 1=> 正常使用 0=>停用此帳號
